# Optical_Interference_And_Newtons_Rings
This repository contains the code used to calculate and present the findings of a laboratory experiment involving thin film interference and light-wave interference through a plano-convex lens. 

# Software
MATLAB R2025a

# Files
- 'NewtonsRings.m' - Main script.
- 'data' - Folder containing all the raw measurements which are imported into MATLAB and processed.
- 'README.md' - This file.
- 
# How to run
- Download files (should automatically download as zip file), extract files. 
- Open MATLAB and locate the folder 'Optical_Interference_And_Newtons_Rings-main' and run the code 'NewtonsRings'. The repository has been structured such that the program will read locally from the 'data' folder.
